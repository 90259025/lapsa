Lapsa: Easy, Elegant Slides

Lapsa is the incredibly simple way to build beautiful and interactive presentations. It uses HTML, CSS, and JavaScript for its layout and functionality, meaning simple things are simple and complicated things are possible.

Use the arrow keys (or enter or space) or tap with two fingers to move forward.

Build and Unbuild with Ease

Move the cursor to the left edge or tap with three fingers to open the shelf, which contains more navigation options.

Lapsa tries to display identically on screens of all sizes, but it works best with aspect ratios that aren’t extremely narrow. If you’re using a phone, try rotating it to landscape!
